# Kata Roman Number

## Installation

```bash
npm install
```

## Exécuter les tests

```bash
npm test
```

## Énoncé

Écrire une fonction permettant de convertir des nombres entiers en **chiffres
romains**.  

### Exemples

- `1` → `"I"`  
- `4` → `"IV"`  
- `9` → `"IX"`  
- `58` → `"LVIII"`  
- `1994` → `"MCMXCIV"`
